★[usaco09novsilver]农场技艺大赛
描述 Description
  农夫约翰有3*N (1 <= N <= 500,000)头牛，编号依次为1..3*N，每头牛都有一个整数值的体重W_i (1 <= W_i <= d)。约翰准备参加农场技艺大赛，向广大的农业社区展示他的奶牛。
大赛规则允许约翰带N头牛参赛。约翰给每头牛赋予了一个“有用度”U_i (1 <= U_i <= h), 它表示了某头牛在比赛中的有用程度。约翰希望他选出的奶牛的有用度之和最大。
有可能选出很多组的N头牛都能达到有用度最大和。约翰害怕选出的N头牛的总重量会给大赛带来震撼，所以，要考虑优先选择体重轻的奶牛。
帮助约翰选出N头总重量最轻，并且有用度之和最大的奶牛。输出体重模M(10,000,000 <= M<= 1,000,000,000)后的余数。
注意：为了使输入更快，约翰使用了一个多项式来生成每头牛的体重和有用度。对每头牛i（0 <= i ＜ 3*N)，体重和有用度的计算公式为：
W_i = (a*i^5+b*i^2+c) mod d
U_i = (e*i^5+f*i^3+g) mod h
各个系数的取值范围为：
0 <= a <= 1,000,000,000;
0 <= b <= 1,000,000,000;
0 <= c <= 1,000,000,000;
0 <= e <= 1,000,000,000;
0 <= f <= 1,000,000,000;
0 <= g <= 1,000,000,000;
10,000,000 <= d <= 1,000,000,000;
10,000,000 <= h <= 1,000,000,000.
这个多项式有时会生成重复的数，你的程序要正确处理这种情况。
输入格式 Input Format
  第1行：10个空格分开的整数: N, a, b, c, d, e, f, g, h, M
输出格式 Output Format
  第1行：满足总重量最轻，且用度之和最大的N头奶牛的总体重模M后的余数。
