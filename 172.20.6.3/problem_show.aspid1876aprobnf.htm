★[usaco]钓鱼岛的护栏
描述 Description
  钓鱼岛要修护栏了，是的，要修护栏了。既然要修护栏，钓鱼岛附近的所有岛屿都要修护栏。
钓鱼岛和它附近的岛我们认为是凸多边形，而要修建的护栏就是多个凸多边形。修护栏的工作交给了小x，小x设计了一个很自认为很优秀的的方案：
1） 他可以从任意一个岛屿的任意一个起点出发，开始修建护栏。
2） 每个岛的护栏都是一个环，每条边必须一次性修建完成。
3） 可以在每个岛的每个顶点，驾驶船到另一个岛的任意顶点，但修建完另一个岛的护栏后，必须回到驾船来之前的那个顶点，在另一个岛上，不能再到其他岛。也就是说，必须要找个起点岛，然后由起点岛屿驾船到其他岛开始修护栏，确定起点岛屿之后，其他岛屿之间不能驾船前往。
4） 任意两个点之间移动都要花费时间，修护栏在一个岛上，不用开船，我们忽略修护栏的花费时间，我们只在乎开船花费的时间。。
5）题目会给你所有的点以及所有的护栏，但是你要算出那些护栏属于同一个岛。
现在小x想知道，他最少花费多少驾船的时间，可以将护栏修建成功。
如果有不理解，看样例解释。
输入格式 Input Format
  第一行一个整数N(3 <= N <= 500)，表示一共有N段护栏要修。
接下来N行，每行两个整数X和Y（1 <= X，Y <= N），表示X与Y之间要修建一段护栏。数据保证肯定在一个且只在一个环内。
接下来N*N的整数矩阵，表示任意两点移动要花费的时间。保证花费的时间不会超过1000.
输出格式 Output Format
  一个整数，最少花费驾船的时间。

样例输出 Sample Output
 
时间限制 Time Limitation
  1s
注释 Hint
 

有一种最优方法是：从3号点出发，到7，1，由1坐船 11，走4,5，回到11，之后再回到1，然后由1再到12，之后2,9,8,12将下方的护栏修好，回到1，之后10,6,3,7.所有工作完成。
驾船经过1到11,11到1,1到12,12到1，时间消耗30.修护栏的时间我们忽略掉。
