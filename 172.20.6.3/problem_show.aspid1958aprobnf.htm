★[usaco2014_jan_bronze]滑雪课程设计
描述 Description
  农夫约翰在他的农场有 N座山（1 <= N <= 1,000） 每一个海拔范围在整数0...100 。
因为在冬天这些山上有很多的雪，农夫约翰经常经营滑雪训练营。

不幸的是，农夫约翰的滑雪训练营因为生意太好，被税务局告知要在明年收税，但是聪明的Fj，根据仔细阅读法律，他发现只要滑雪场最高山与最低山的高度差别不大于17，他就可以避免交税。
当然，改造山是要有花费的，Fj决定要一劳永逸的改造山而不交一分钱税。现在Fj知道如果改变山的高度x个单位，他要花费x^2单位的资金，现在Fj想知道，最少花多少钱可以完成这个计划（他改变山的高度只是整数）
输入格式 Input Format
  * 第一行:  一个整数N（1<=N<=1000）.

* 第二到N+1行: 每行包括一座山的海拔.
输出格式 Output Format
  一行: FJ需要支付给修改山的海拔的最小值使修改山后的最大山和最小山差最多17个单位.
