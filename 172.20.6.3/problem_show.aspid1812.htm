☆[8.21]逃离游戏
描述 Description
  小x在玩一个逃跑游戏，游戏的规则如下：
小x控制的人物从N层楼上开始往下跑，跑到1层游戏结束，每层的逃离速度都有一个下限Vi，小x逃离的速度必须不小于Vi，且不能超过Vi+k，
大楼会在固定时间倒塌，如果太慢，就不能安全逃离，如果太快，大楼会承受不了，会提前倒塌。
小x控制的人物初始速度为V，每下一层楼，小x可以让控制的人物的速度+1，或-1，或不变。
现在小x想知道，能否顺利过关，如果能，求出速度的最大平均值，即（第n层到第2层的速度之和）/（N-1） 的最大值
提示：1楼不需要再下了，N层楼只需要下N-1层。并且在第N层楼到N-1层时必须为初始速度。
输入格式 Input Format
  第一行，三个整数N（楼的层数），V（初始速度），K（速度上限的常数）
第二行，N-1个整数，Vi表示第i+1层楼到第i层楼速度的最小值
输出格式 Output Format
  若能下楼，输出速度之和除以（N-1），保留两位小数。
若不能，宣扬一下国策，输出“YI DING YAO JIAN CHI JI HUA SHENG YU”（不含引号）
样例输入 Sample Input
 
样例输出 Sample Output
 
时间限制 Time Limitation
  1s
注释 Hint
  【样例解释】
样例一：3楼到2楼，速度为初始的3
　　　　  2楼到1楼，速度加快，为4
　　　　  输出（3＋4）/2=3.50
样例二：3楼到2楼，速度为初始的3，3－（2+0）>0,速度太快，大楼直接倒塌。
【数据范围】
  2〈=N〈=100，0〈=K〈=100，1〈=V,Vi〈=100。
